# Installation

## Requirements

- Python >= 3.10
- NumPy >= 1.20.0
- Numba >= 0.56.0
- SciPy >= 1.7.0

## Install

```bash
pip install -r requirements.txt
```

## Development Setup

```bash
pip install -r requirements.txt
pip install ruff pytest
```

## Verify Installation

```python
from nchash import run_hash
print("NCHASH installed successfully!")
```
